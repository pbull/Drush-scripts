redirect_import.drush.inc
* Additional Drush command for importing redirects from a CSV file or by parsing the Redirect entries in a .htaccess file. For use with the Redirect module for Drupal 7 (http://drupal.org/project/redirect).

path_redirect_import.drush.inc
* Additional Drush command for importing redirects from a CSV file or by parsing the Redirect entries in a .htaccess file. For use with the Path Redirect module for Drupal 6 (http://drupal.org/project/path_redirect).
